# Contributing

Thanks for considering a contribution.

## Good Contributions

- New repository health checks
- More precise security-warning patterns
- Documentation examples
- Tests for edge cases
- CLI output improvements

## Local Setup

```bash
npm install
npm run lint
npm test
npm run build
```

## Pull Requests

Keep changes focused. Include tests for scanner behavior and update docs when output changes.

## Security

Do not include real tokens, private endpoints, logs with credentials, or private repository data in issues or pull requests.
